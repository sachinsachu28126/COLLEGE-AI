# CampusAI - College FAQ Assistant

An AI-powered FAQ chatbot for colleges built using Flask and Sentence Transformers.

## Features
- Answers college FAQs
- Semantic search using AI embeddings
- User-friendly interface
- Supports admissions, courses, hostel, placements, and facilities queries

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- Sentence Transformers
- Scikit-learn
- JSON

## Run the Project
pip install -r requirements.txt
python app.py
